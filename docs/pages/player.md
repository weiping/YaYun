# 播放页面设计文档

## 页面概述

播放页面是雅韵App的核心体验页面，以高度沉浸式的方式模拟黑胶唱片播放体验，展示当前播放音乐的详细信息，同时提供直观的播放控制功能。该页面摒弃了常规导航栏和播放条，而是将整个屏幕空间用于创造一个专注的音乐欣赏环境。

## 设计目标

- 创造沉浸式的古典音乐欣赏体验
- 通过黑胶唱片模拟增强视觉趣味性
- 提供高质量音频选项和直观控制
- 支持深色/浅色主题切换
- 添加音频可视化效果增强感官体验

## 页面结构

### 顶部区域

1. **返回按钮**
   - 位于左上角，圆形设计
   - 包含向下箭头图标，指示向下滑动返回
   - 支持按压反馈效果

2. **更多操作按钮**
   - 位于右上角，圆形设计
   - 包含竖向三点图标
   - 点击可切换深色/浅色模式

### 唱片播放区域

1. **唱针组件**
   - 位于唱片右上方
   - 播放时移动到唱片上方，暂停时抬起
   - 动画过渡使用缓动函数提供平滑效果

2. **黑胶唱片**
   - 大尺寸圆形元素，占据屏幕主要部分
   - 质感设计，包含:
     - 黑色唱片主体
     - 同心圆纹理模拟唱片沟槽
     - 中央标签显示专辑封面
     - 表面反光效果增强真实感
   - 播放状态下持续旋转动画
   - 支持暂停与播放状态切换动画

3. **音频可视化**
   - 位于唱片下方，32个频谱条
   - 播放时根据节奏动态变化高度
   - 可通过底部按钮切换显示/隐藏

### 音乐信息区域

1. **歌曲标题**
   - 使用衬线字体增强古典感
   - 字体略大，强调重要性
   - 居中排列

2. **演奏者/指挥信息**
   - 次要信息，使用较小字号
   - 颜色对比度降低，形成层次感
   - 根据主题自动调整颜色

### 控制区域

1. **进度条**
   - 线性进度显示，位于歌曲信息下方
   - 带有可拖动的进度把手
   - 显示当前播放时间和总时长

2. **播放控制按钮**
   - 五个控制按钮组合:
     - 上一曲
     - 快退
     - 播放/暂停(品牌色突出显示)
     - 快进
     - 下一曲
   - 播放/暂停按钮尺寸更大，位置居中

3. **音质选择器**
   - 三种音质选项:标准音质、高品质、无损
   - 当前选中项使用品牌色高亮
   - 点击时显示提示信息与涟漪效果

4. **底部功能按钮**
   - 四个辅助功能:
     - 可视化开关
     - 分享
     - 收藏/喜欢
     - 评论
   - 排列紧凑，带图标和文字标注

## 交互设计

1. **唱片动效**
   - 播放状态:唱片匀速旋转，唱针放下
   - 暂停状态:唱片停止旋转，唱针抬起
   - 切换歌曲:唱针抬起，唱片停止，切换后重新开始

2. **粒子效果**
   - 所有可点击元素点击时触发
   - 品牌色粒子从点击位置向外扩散
   - 粒子大小随机，增强视觉反馈

3. **音质选择交互**
   - 点击切换音质选项
   - 触发涟漪动画和提示信息
   - 显示关于音质特性的简要说明

4. **音频可视化**
   - 随音乐变化，模拟音频频谱
   - 可通过底部按钮切换显示/隐藏
   - 不播放时保持低振幅状态

5. **进度控制**
   - 支持拖动进度条调整播放位置
   - 实时更新时间显示
   - 大小适合触摸操作

## 视觉设计

1. **配色方案**
   - 深色模式:深黑背景(#121212)，白色文字，品牌红色(#8B0000)强调
   - 浅色模式:白色背景(#FFFFFF)，深色文字，品牌红色强调保持一致
   - 专辑封面作为背景模糊效果，增加沉浸感

2. **排版层次**
   - 使用标题（衬线字体）、正文和辅助文字三级层次
   - 控制字重和颜色形成视觉引导
   - 关键元素使用品牌色加强关注度

3. **动效设计**
   - 所有过渡使用平滑动画
   - 适当使用弹性缓动增强交互感
   - 控制动画速度保持优雅且不分散注意力

## 响应性设计

1. **不同设备适配**
   - 页面占满整个可用空间，充分利用屏幕
   - 元素尺寸和间距保持良好比例
   - 唱片区域在更大屏幕上放大，但保持圆形比例

2. **横屏/竖屏切换**
   - 竖屏为主要设计布局
   - 横屏时调整元素位置，保持易用性

## 无障碍设计

1. **操作区域大小**
   - 所有可点击元素至少44×44px，便于触摸操作
   - 按钮间距充足，防止误触

2. **主题适配**
   - 深色/浅色主题确保在不同环境下的可读性
   - 所有文本符合WCAG对比度要求

## 核心交互流程

1. **播放控制**
   - 点击播放按钮 → 唱针下降 → 唱片开始旋转 → 进度条开始移动
   - 点击暂停按钮 → 唱针抬起 → 唱片停止旋转 → 进度条停止
   - 更换歌曲 → 唱针抬起 → 唱片停止 → 信息更新 → 新唱片旋转 → 唱针下降

2. **主题切换**
   - 点击右上角"更多"按钮 → 触发粒子效果 → 切换深色/浅色主题
   - 所有元素颜色平滑过渡到新主题

3. **音质切换**
   - 点击音质选项 → 切换高亮状态 → 显示提示信息 → 涟漪动画效果

4. **返回操作**
   - 点击左上角返回按钮 → 触发粒子效果 → 返回上一页面
